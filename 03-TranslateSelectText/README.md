## Translate Select Text

1,翻译API-扇贝
2,仅适用于支持fetch方法的浏览器
![](images/fetchsupport.png)

## 安装方法
第一种
	下载本项目

	打开chrome浏览器“扩展程序”，勾选上“开发者模式”开启开发者模式，点击“打包扩展程序”，选中到刚刚下	载的目录，生成.crx插件

	将生成的插件拖到“扩展程序”中即可

第二种
	打开chrome浏览器“扩展程序”，勾选上“开发者模式”开启开发者模式，点击“加载已解压的扩展程序...	  ”，选择下载代码解压的文件夹即可。
